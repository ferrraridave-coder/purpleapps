Frequently Asked Questions

What is Microscope AI?

Microscope AI is a tool for exploring and understanding AI models. It lets you browse model information, save your own model profiles, and view AI-generated analysis of traits, quirks, and potential biases. You can compare models side by side and estimate usage costs using the built-in cost calculator.

Do I need an account or API keys?

Microscope AI does not require you to create an account and does not ask you to enter any API keys. The app uses a secure backend service to talk to providers such as Groq for analysis, using the developer's own credentials. Public model information is retrieved from the web (including sources such as OpenRouter) without needing user accounts or API keys.

Is there a subscription?

Microscope AI is a standalone app available on the App Store. There are no in-app subscriptions or usage-based charges from the app itself. Any prices shown in the app are estimates of third-party model costs and are for reference only — they are not charges made by Microscope AI.

What models are supported?

Microscope AI focuses on models that are publicly listed by providers such as OpenRouter and other documented model catalogues. You can create and save profiles for a wide range of language models, including general-purpose, "thinking"/long-context, and specialised models, as long as their details are publicly available.

Does Microscope AI send my data to AI providers?

When you request an analysis for a model, relevant text (for example, the model name and any description used for context) is sent securely to the backend service, which then forwards it to providers such as Groq to generate the summary, traits, quirks, and bias information. Microscope AI does not store your analyses on the device beyond the profiles you choose to save, and it does not store API keys. For details on how third-party providers handle data, please refer to their own privacy policies.
